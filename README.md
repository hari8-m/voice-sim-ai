# VoiceSim AI – Real-Time Voice-Based Scenario Learning Assistant

## 📌 Problem
Many students struggle to practice real-life communication like interviews and daily conversations due to lack of real-time voice interaction.

## 💡 Solution
VoiceSim AI is a real-time voice assistant using Murf AI that allows users to practice real-life scenarios through voice interaction.

## 🚀 Features
- Real-time voice interaction
- Multilingual support
- Scenario-based learning (interview, daily conversation)
- AI voice responses using Murf API

## 🛠 Tech Stack
- HTML
- JavaScript
- Murf AI API
